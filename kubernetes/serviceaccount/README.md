# ServiceAccount 操作

### 创建

创建 ServiceAccount 可用字段：
- Namespace：必填
- Name：必填
- Label：可选

### 列表

指定命名空间列出 ServiceAccount：
- Namespace：必填
- Label：可选

列出所有命名空间的 ServiceAccount：
- Label：可选

### 查询

查询指定命名空间的 ServiceAccount：
- Namespace：必填
- Name：必填

### 删除

删除指定命名空间的 ServiceAccount：
- Namespace：必填
- Name：必填
