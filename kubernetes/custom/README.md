# 自定义资源操作

### 可用参数

- Kind（资源类型）
- Method（操作方法）
- Name（资源名称）
- Namespace（命名空间）
- JsonData（JSON 数据）

### 列表

列出自定义资源：
- Kind：必填
- Method：必填
- Namespace：按命名空间过滤时必填

### 查询

查询自定义资源：
- Kind：必填
- Method：必填
- Name：必填
- Namespace：命名空间资源时必填

### 删除

删除自定义资源：
- Kind：必填
- Method：必填
- Name：必填
- Namespace：命名空间资源时必填

### 创建

创建自定义资源：
- Kind：必填
- Method：必填
- JsonData：必填
